# SoundSphere - Персональная музыкальная коллекция

Каталогизатор музыки с интеграцией Spotify API, созданный на React, Redux, TailwindCSS и Skeleton.dev.

##  Функционал

- **Поиск по Spotify API** - Поиск треков через Spotify API с fallback на mock данные
- **Создание плейлистов** - Создавайте и управляйте своими плейлистами
- **Рейтинги и теги** - Оценивайте треки и добавляйте теги для организации
- **Статистика прослушиваний** - Отслеживайте количество прослушиваний и популярные треки
- **Рекомендации** - Персонализированные рекомендации на основе ваших оценок и прослушиваний
- **Офлайн-режим** - Все данные сохраняются в IndexedDB для работы без интернета

##  Технологии

- **React 18** - UI библиотека
- **Redux Toolkit** - Управление состоянием
- **React Router** - Маршрутизация
- **TailwindCSS** - Стилизация
- **IndexedDB (idb)** - Офлайн хранилище
- **Vite** - Сборщик

##  Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev сервер:
```bash
npm run dev
```

3. Откройте браузер на localhost
